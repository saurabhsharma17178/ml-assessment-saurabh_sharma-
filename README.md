- Used Decision Tree, Random Forest, and Gradient Boosting
- Performed preprocessing, scaling, and evaluation

### Q2: Unsupervised Learning

- Used Elbow method to determine optimal clusters
- Applied PCA for visualization

### Q3: Feature Engineering & Regression
- Created date-based features
- Built ML pipeline using ColumnTransformer
- Trained Linear Regression and Random Forest models
- Evaluated using RMSE and MAE
